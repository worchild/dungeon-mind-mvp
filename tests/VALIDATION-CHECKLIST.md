# Manual Validation Checklist

Before tagging a release:

- [ ] Game loads on GitHub Pages
- [ ] Validator says state OK
- [ ] Starting room is R01
- [ ] Movement works from R01 to R02
- [ ] Search reveals a clue
- [ ] Loot adds item to inventory
- [ ] Monster defeat button works in monster rooms
- [ ] Threat meter increases on first room entry where configured
- [ ] Save works
- [ ] Load works
- [ ] Export creates JSON
- [ ] Import accepts valid exported JSON
- [ ] Reset returns to initial state
- [ ] Player-visible text does not reveal hidden DM notes
